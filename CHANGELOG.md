# Changelog

## v2.0.0 (2026-03-27)

### 規則修正
- **路線網路全面校正**：1條運河限定 / 8條鐵路限定 / 32條兩者皆可
- **收入軌重寫**：變速間距101格（低等級1格/級，高等級4格/級）
- **起始收入 £0**（軌道格10）
- **鐵路時代首輪2行動**（只有運河時代首輪才是1行動）
- **牌數精確校正**：2人=40張 / 3人=54張 / 4人=64張
- **煤炭市場購買需連接商人**圖標
- **商人需蓋路才能販賣**（不再自動連接）
- **陶瓷 Lv1/3/5 不能研發**（💡燈泡標記）
- **農莊啤酒廠限制**：只能用啤酒產業牌建造
- **金錢不能低於0**：建造前先計算總費用
- **借貸下限 -10**：收入退3級後不能低於-10
- **手牌用完自動跳過**
- **行動順序**：改為本輪花費最少者先行動
- **商人板塊隨機**：每場接受的產業不同
- **商人啤酒獎勵**：免費研發改為彈窗選擇
- **萬能牌修正**：偵查獲得的萬能牌可正常使用
- **產業牌網路限制**：修正只有路線沒有建築時的判斷
- **煤/鐵方塊回到市場**：從板塊消耗後正確回到市場

### UI 改版
- **實際遊戲板底圖** + HD 放大（2000x2000）
- **卡牌圖片支援**：手牌顯示插畫背景
- **Steam 風格產業面板**：每個板塊獨立小卡，hover 放大
- **煤/鐵市場**：黑色/橘色點點直接疊在底圖上
- **商人詳細 tooltip**：hover 顯示接受類型、啤酒、獎勵
- **縮放平移**：滾輪縮放 + 右鍵拖拽 + 雙擊重置
- **玩家顏色邊框**：翻面後仍能辨識板塊所有者
- **斷線重連**：重新整理自動回到遊戲
- **回合通知**：瀏覽器推播 + 音效 + 標題閃爍
- **自動跳過按鈕**：一鍵持續跳過
- **詳細行動記錄**：每步花費、市場變化、收入結算
- **格子獨立校正工具**：每個格子可獨立調整位置和產業類型

### 工具
- `/calibrate.html` — 城市位置校正
- `/calibrate-slots.html` — 格子獨立校正（含大小、間距、產業類型）

## v1.0.0 (2026-03-26)

### 初始版本
- 2-4人即時線上對戰（Socket.io）
- 完整兩個時代（運河 + 鐵路）
- 6種行動：建造 / 建路 / 研發 / 販賣 / 貸款 / 偵查
- 6種產業：棉花 / 煤礦 / 鐵廠 / 工廠 / 陶瓷 / 啤酒廠
- Canvas 地圖渲染
- 大廳系統（建立/加入房間）
- Cloudflare Tunnel 公開連線
- 全中文介面
