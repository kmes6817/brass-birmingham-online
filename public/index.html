🎮 基本流程

每位玩家每回合有 2 個行動（第一輪各1個）。行動後棄掉對應的牌。

🏭 建造

打出城市牌 → 在該城市建；打出產業牌 → 在網路上任何地方建。消耗煤/鐵資源。

🛤 建路

運河時代：每條路消耗 £3 + 棄1張牌。鐵路時代：每條路 £5 + 1煤 + 棄1張牌。

⬆ 研發

花費 £1 + 鐵×1～2，移除最低等級的產業板塊（升級到下一級）。

💰 販賣

連接到商人 → 翻轉相連的棉花/陶器板塊，取得 VP 和收入。需要商人有啤酒。

🏦 貸款

取得 £30，收入軌道降低 3 格。

🔭 偵查

棄3張牌，取得2張萬用牌（地點萬用 + 產業萬用）。

📊 計分時機

運河結束：所有連結VP + 所有已翻轉建築VP。清除所有板塊。
鐵路結束：同上，加上連結VP × 兩側翻轉建築。

⌨ 鍵盤快捷鍵

B 建造 N 建路 D 研發 S 販賣 L 貸款 C 偵查 P 跳過 Enter 確認 Esc 取消
